Abstract — Brain Tumor Segmentation
Accurate and quick brain tumor diagnosis and treatment are critical for successful management. This work describes a novel technique using three deep learning architectures: GlobalNet, Multi-task Learning, and FusionNet. GlobalNet incorporates global context via a global pooling layer; Multi-task Learning allows concurrent tasks sharing knowledge; FusionNet combines information from multiple CNNs trained on various medical imaging modalities to improve accuracy and resilience of brain tumor segmentation.
Citation
S. Shreyanth, S. Niveditha and V. Kathiroli, "Accurate Brain Tumor Segmentation and Detection using Multi-Task Learning with GlobalNet and FusionNet," 2023 IEEE 12th International Conference on Communication Systems and Network Technologies (CSNT), Bhopal, India, 2023, pp. 478-485, doi: 10.1109/CSNT57126.2023.10134722.
Abstract — Speech Signal Segmentation
Speech signal segmentation is a key component of speech processing tasks including speaker recognition and emotion recognition. We propose a novel approach using kernelized deep networks with MFCC and LPC for feature extraction, spectral subtraction and Wiener filtering as preprocessing, PCA and LDA for dimensionality reduction, and spectral clustering using the graph Laplacian. Results on TIMIT and BURN corpus demonstrate superior segmentation accuracy with reduced computational complexity.
Citation
S. Niveditha, S. Shreyanth, V. Kathiroli, P. Agarwal and S. Ram Abishek, "Kernelized Deep Networks for Speech Signal Segmentation Using Clustering and Artificial Intelligence in Neural Networks," 2023 IEEE 12th International Conference on Communication Systems and Network Technologies (CSNT), Bhopal, India, 2023, pp. 667-674, doi: 10.1109/CSNT57126.2023.10134609.
Abstract — Cluster-Based Grid Computing
Grid computing based on clusters improves efficacy of wireless network data transmission. This study examines cluster-based grid computing, routing analysis protocols, and deep learning to optimize data transmission. The proposed method utilizes clusters to distribute computing duties, a novel routing analysis protocol that dynamically adapts to network conditions, and deep learning for intelligent data routing. Results show substantial enhancements in throughput, latency, and energy consumption.
Citation
S., Shreyanth & S., Niveditha. (2023). CLUSTER-BASED GRID COMPUTING ON WIRELESS NETWORK DATA TRANSMISSION WITH ROUTING ANALYSIS PROTOCOL AND DEEP LEARNING. International Journal of Advanced Research. 11. 517-534. 10.21474/IJAR01/17096.
Abstract — Neural Image Caption Generation
We propose a novel model for neural image caption generation with visual attention to help visually impaired individuals access online images. Our model uses CNNs and RNNs to convert image content into aural descriptions. The model operates in two phases: a text-to-speech API converts image content into textual description, then the extracted description is converted to audio. This research promotes digital inclusion and equality for millions of visually impaired people.
Citation
Priyanka Agarwal, Niveditha S, Shreyanth S, Sarveshwaran R, Rajesh P K, "Neural Image Caption Generation with Visual Attention: Enabling Image Accessibility for the Visually Impaired," International Journal of Scientific Research in Science, Engineering and Technology (IJSRSET), Print ISSN: 2395-1990, Online ISSN: 2394-4099, Volume 10, Issue 3, pp.562-575, May-June-2023.
Abstract — Machine Learning in VLSI Design
Machine learning has made an impact in the microchip area and will eventually supplant current VLSI design concepts. Design creation has been automated by substituting time-consuming traditional concepts developed by experts. Several major firms including IBM and Intel have their own CAD division for design automation. Abstract — Cerebrovascular Accident Prognosis
This study compares the predictive performance of three supervised learning algorithms — Random Forest, Decision Tree, and XGBoost — for cerebrovascular disease prediction. Random Forest achieved the highest prediction accuracy of 99.93%, XGBoost showed 97.94%, and Decision Tree obtained 97.53%. These results outperform state-of-the-art methods and demonstrate the potential of supervised learning for accurate cerebrovascular prediction in clinical practice.
Citation
B. Sethuraman and S. Niveditha, "Cerebrovascular Accident Prognosis using Supervised Machine Learning Algorithms," 2023 World Conference on Communication & Computing (WCONF), RAIPUR, India, 2023, pp. 1-8, doi: 10.1109/WCONF58270.2023.10235122.
Abstract — Malware Classification
This study presents a unique method for predicting malware category and family using machine learning, leveraging the Cuckoo environment and automated feature selection via the Boruta method. The KNN technique was combined with dynamic analysis of malware samples within the Cuckoo environment. Test Case 3 (Automated Feature Selection + Cross-Validation) showed outstanding specificity (90%), precision (93%), recall (96%), and F1-Score (92%) for phishing detection.
